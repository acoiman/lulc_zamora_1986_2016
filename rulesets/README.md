This folder contains  [eCognition](https://geospatial.trimble.com/products-and-solutions/ecognition) rule sets applied for classifying [Landsat](https://landsat.gsfc.nasa.gov/) images in Zamora municipality, Aragua state, Venezuela, from 1986 to 2016 following the [Corine Land Cove](https://land.copernicus.eu/pan-european/corine-land-cover)r schema.
