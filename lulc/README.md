This folder contains the results in [shapefile](https://en.wikipedia.org/wiki/Shapefile) and  [geojson](https://geojson.org/) format of [Landsat](https://landsat.gsfc.nasa.gov/) image classification in Zamora municipality, Aragua state, Venezuela, from 1986 to 2016 following the [Corine Land Cove](https://land.copernicus.eu/pan-european/corine-land-cover)r schema.
